# CANDY
Personal Project Repository

## Project Structure

* **BROWSER/**: Contains browser-side logic, likely handling frontend automation or rendering for the scraper.
* **WRAPPER/**: Contains the core wrapper logic for the scraping engine.

## Features

* **Wrapper Scraper**: specialized logic for wrapping and parsing web content.
* **Fuzzy Search**: Implements fuzzy matching algorithms to find relevant data even with partial or approximate queries.
* **Link Extraction**: Automatically identifies and extracts hyperlinks from target pages.

## Tech Stack
* **Python** (Core logic)
* **JavaScript / HTML / CSS** (Browser interaction and interface)
